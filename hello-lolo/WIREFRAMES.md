# Hello, Lolo — Wireframes

Detailed ASCII wireframes for all key screens. Designed for mobile-first (375pt width).
Color references use the brand palette: sunset orange, warm yellow, deep green, cream white.

## Design Principles

1. **Tagalog first, English second** — Headers, buttons, and labels use Tagalog with English subtitles (teaches passively)
2. **Every screen teaches** — Even navigation labels, greetings, and error states use Tagalog
3. **Warmth over gamification** — Progress feels like building a family, not grinding points
4. **Humor is cultural** — Filipino humor (self-deprecating, family-teasing) woven throughout
5. **Audio everywhere** — Every Tagalog word on every screen should be tappable for pronunciation
6. **Three registers always available** — Formal / Casual / Taglish toggle is global, not per-lesson
